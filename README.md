# nemostore

네모스토어 데이터 수집 및 분석 프로젝트입니다.

## 프로젝트 구조
- `raw/`: 수집 원본 데이터 (CSV, JSON 등)
- `data/`: 전처리 완료된 데이터 (SQLite DB 등)
- `images/`: 시각화 결과물 (그래프, 차트)
- `scripts/`: 데이터 수집 및 분석 코드 (.py)
- `plan/`: 수집 계획 및 프롬프트 정의
- `report/`: 분석 결과 보고서
- `presentation/`: 발표 자료

## 참여 방법
- 가상환경은 상위 폴더의 `.venv`를 사용합니다.
- 패키지 설치는 `uv pip install`을 이용해 주세요.
